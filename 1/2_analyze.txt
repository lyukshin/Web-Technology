Какие блоки находятся на главное страницы сайте ru.wikipedia.org

1. Шапка сайта
В шапке сайта находится меню статуса пользователя, размещаем в правом верхнем углу, пункты: статус пользователя (авторизован или нет) далее идут пункты меню  (Обсуждение Вклад Создать учётную запись Войти)

2. Навигационнная часть сайта, размещаем с левой стороны, идет во всю длинну страницы
В верхней части размещаем логотип компании, под ним будет находится навигация по сайту
Первый блок навигая по статьям
Второй блок "Участие" ссылки на страницы где пользователь сможет взаимодействовать с сайтом
Третий блок "Инструменты" ссылки на технические страницы
Четвертый блок "Печать экспорт" отвечает за печать или экспорт страницы
Пятый блок "В других проектах" выводим список других продуктов компаниии
Шестой блок "На других языках" Выводим список языковых версий сайта

3. Зона контента
В верхней части блока контента размещаем функционал для взаимодействия со страницей, делим на две части, правую и левую
В левой будет находится меню пункты взаимодействия, а в правой части будет находится подменю для этих пунктов и строка поиска
Пункты взаимодействия
1. Первый пункт название "Заглавная", выбран по умолчанию, вкладка показывает контент страницы
Пункты подменю:
1.1 Читать - Название ссылки (Читать), Ссылка ведет на (url ссылки), выбрана по умолчанию, при выборе в контенте отображаются основные блоки контента
1.2 Просмотр кода - Название ссылки (Просмотр кода), Ссылка ведет на (url ссылки), при выборе отображается информационные блок, под ним редактор кода для этой страницы
1.3 История - Название аналогично п1. выводим фильтр правок, и список изменений для страницы на которой находимся

2. "Обсуждение" при клике на нее пользователь получает список страниц где он может обсудить статью
2.1 Используется аналогично п 1.1
2.2 Править код - При клике попадаем в раздел для правки кода
2.3 Добавить тему - Блок позволяет пользователю добавить тему для публикации
2.4 История - Работает аналогично п 1.3

Структура контента
Под меню размещает тег H1 
Далее размещаем блоки
3.1 Избранные статьи
3.2 Недавно добавленные статьи
3.3 Блоки статей по рубликам "География, Наука, История, Экономика, Культура и общество и т.д."
Далее в низу размещаем меню с категориями

4. Подвал сайта
В подвале сайта размещаем дату последнего редактирования сайта
Лицензионное соглашение
Меню с юредическими документами и кнопки с логотипами продуктов, которые использовались в системе управления сайтом
